# To-Do Notes

A full-stack CRUD application built with:

- TypeScript
- React
- Node.js
- Express
- MySQL
- React Hook Form

## Features

- Create users
- Update users
- Delete users
- Create to-do lists for each user
- Update to-do lists
- Delete to-do lists

## Installation

Backend:
- npm install
- npm run dev

Frontend:
- cd frontend
- npm install
- npm run dev
